# API de Gerenciamento de Tarefas

Projeto fullstack de gerenciamento de tarefas (to-do list), desenvolvido com **Django REST Framework** no backend e **React + TypeScript** no frontend.  
O objetivo principal é reforçar de forma simples conceitos estudados e aplicar conceitos novos de forma a aprender melhor eles

## Tecnologias utilizadas
### Backend
- Python
- Django REST Framework
- Pytest (testes automatizados)
- Docker

### Frontend
- React + TypeScript
- Vite
- TailwindCSS + shadcn/ui
- Lucide Icons
- Jest (testes)

## Funcionalidades
- Cadastro e autenticação de usuários (JWT)
- CRUD de tarefas (criar, listar, atualizar, excluir)
- Organização de tarefas por usuário
- Interface web responsiva para gerenciamento
